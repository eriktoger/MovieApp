My first attempt to create a app using React Native and Expo.

The app connects to the backend through the baseUrl in Variables.js:

```javascript
export default ENV = {
  dev: {
    baseUrl: "url-to-backend",
  },
  staging: {
    baseUrl: "url-to-backend",
  },
  prod: {
    baseUrl: "url-to-backend",
  },
};
```
